## Svelte 3

### Svelte component

```.js
<div>
	<h2>Array Demo</h2>

	{#if items.length > 0}
	<ul>
		{#each items as item, index}
		<li>{item.title}</li>
		{/each}
	</ul>
	{/if}
	<button on:click={fetchMore}>Fetch More</button>
</div>
  
<script>
  let items = [{ title: "Item 1" }, { title: "Item 2" }];
  function fetchMore() {
    items = [...items, { title: "Fetched item" }];
  }
</script>
  
<style>
  h2 {
    color: darkblue;
  }
</style>
```
